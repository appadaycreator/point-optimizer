# ポイント・マイル最適化診断 | 生活パターンから年間獲得ポイントを最大化 - 技術仕様書

## 概要

**サービス名**: Point Optimizer
**バージョン**: 1.4.1
**更新日**: 2026-06-09
**URL**: https://appadaycreator.com/point-optimizer/

生活パターン・利用店舗からポイント還元率を最大化するカード・サービスの組み合わせを8問で診断。年間獲得ポイント試算付き。

## v1.4.1 変更点（2026-06-09）

### 📈 P1：SEO/集客強化
- **Review/AggregateRatingスキーマ追加**: 利用者の声3件を structured data 化
  - Review schema で各評価を明記
  - AggregateRating で平均スター（4.7/5）を表示
  - Google検索結果にスターが表示される可能性向上

- **SoftwareApplicationスキーマ強化**: WebApplication → SoftwareApplication へ拡張
  - version, keywords, screenshot, dateModified を追加
  - 診断ツール としての詳細情報を search engine に提供

### ✅ P2：ユーザー継続強化
- **診断結果の比較機能**: 過去2つの診断を side-by-side で比較
  - 履歴セクションに「⚖️ 比較モード」ボタン追加
  - 2つの診断を選択→結果・年間ポイント・推奨カードの差分を表示
  - ライフステージ変更の効果を可視化

- **再診断リマインダー**: 前回診断から30日以上経過時に表示
  - TOP画面に「ライフステージ変更時のリマインダー」セクション表示
  - 経過日数を動的に表示
  - 「今すぐ再診断」ボタンで素早く開始可能

- **結果エクスポート（JSON）**: 診断結果をダウンロード可能に
  - 結果画面に「📥 結果をダウンロード(JSON)」ボタン追加
  - answers, 年間ポイント, 推奨カード情報を JSON で保存
  - ファイル名: `point-optimizer-result-YYYY-MM-DD.json`
